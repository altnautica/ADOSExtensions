# Hello Hybrid Plugin

Skeleton hybrid plugin generated by `create-ados-plugin`. Has both an
agent half (Python subprocess) and a GCS half (TypeScript bundle in a
sandboxed iframe).

## Develop

```sh
pnpm install
pnpm test
pnpm build
uv pip install -e agent
```

## Communication between halves

The agent half publishes events on the host event bus. The GCS half
subscribes via `ctx.client.on('event.<name>', ...)` once the operator
grants the appropriate permissions. See
`https://docs.altnautica.com/developers/event-hooks`.

## Pack and sign

Use `scripts/pack.sh` and `scripts/sign.sh` from
`altnautica/ADOSExtensions` to produce a signed `.adosplug`.
