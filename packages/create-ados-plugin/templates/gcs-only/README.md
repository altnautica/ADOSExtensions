# Hello Plugin

Skeleton GCS-only plugin generated by `create-ados-plugin`.

## Develop

```sh
pnpm install
pnpm test          # vitest unit tests
pnpm build         # produces gcs/plugin.bundle.js
```

## Pack and sign

The repo-level `scripts/pack.sh` and `scripts/sign.sh` (from
`altnautica/ADOSExtensions`) can be vendored into your own project
or you can write your own. The .adosplug archive must contain the
manifest, the bundle, the locale files, and the asset hashes the
manifest declares.

## Install in Mission Control

Drag the signed `.adosplug` into Mission Control -> Settings ->
Plugins -> Install. Approve the manifest preview, grant the requested
permissions, and the panel mounts in the FC tab.

See `https://docs.altnautica.com/developers/quickstart` for the full
walkthrough.
