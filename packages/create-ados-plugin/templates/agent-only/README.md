# Hello Agent Plugin

Skeleton agent-only plugin generated by `create-ados-plugin`. Runs as
a subprocess under `ados-supervisor` on the drone agent.

## Develop

```sh
uv pip install -e agent
```

Talk to the host over the UDS msgpack bridge that the supervisor opens
on startup. See `https://docs.altnautica.com/developers/sdk-python`.

## Pack and sign

Use the repo-level `scripts/pack.sh` and `scripts/sign.sh` from
`altnautica/ADOSExtensions` to produce a signed `.adosplug`. Install
on the agent with `ados plugin install <file>.adosplug`.
